# Pac-Man Game

This is a simple Pac-Man game created with HTML, CSS, and JavaScript.

## How to Play

1.  Clone this repository to your local machine.
2.  Open the `index.html` file in your web browser.
3.  Use the arrow keys to move Pac-Man.
4.  Eat all the dots in the maze to win the level.
5.  Avoid the ghosts!

## Development

This project does not have any dependencies to install. Simply open the `index.html` file in a browser to run the game.
